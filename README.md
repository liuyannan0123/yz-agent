# yz-agent

Local AI knowledge assistant with RAG, OCR preview, and fast-answer routing.

## Open Source

This repository is released under the Apache-2.0 license.

This is the public Git edition of the project. It keeps:

- public documentation
- product plan
- service shell files
- OCR sample script
- startup script examples

It does not include:

- full private knowledge-QA implementation
- full hybrid retrieval logic
- full fast-answer rules
- OCR enhancement logic
- structured extraction internals
- private business rules
- real knowledge-base data
- passwords, logs, or model cache
