## 当前产品方向

项目当前的核心方向是：

1. 上传解析层
- 文件上传
- OCR
- 提取预览
- 预览可编辑
- 确认入库
- 同文件覆盖更新

2. 混合检索层
- 中文 query 拆分
- 标题/文件名召回
- 关键词召回
- 向量召回
- 文件级重排
- 低置信度拒答

3. 结构化抽取层
- 简历类
- 工作日志类
- 流程/制度类

4. 回答路由层
- 快答优先
- 总结类问题再走模型

## 快答机制说明

快答的核心是：

- 先命中知识库
- 再直接抽字段或拼步骤
- 不走慢速模型长生成

也就是说：

- 快答 = 检索/抽取/模板返回
- 慢答 = 检索 + 模型生成
